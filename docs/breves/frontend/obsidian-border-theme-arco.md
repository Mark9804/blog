# Obsidian Border 定制主题：Arco

基于 [Akifyss/obsidian-border](https://github.com/Akifyss/obsidian-border) 的 Arco Design 风格 Obsidian 主题。

![预览](https://cdn.sa.net/2024/12/17/zkDUIFmdWlTCVsL.webp)

## 特色

在 Border 主题基础上调整了部分样式，并使用 Arco Design 的配色方案。

- **元素对齐**：严格对齐标签页元素与主卡片元素，整体布局更加简洁。
- **卡片阴影**：基于 Arco Design 卡片阴影样式重新设计，阴影更加柔和，视觉干扰更低，主界面更突出。
- **色卡优化**：使用 Arco Design 色卡，整体更加清爽。
- **样式微调**：增加行间距，加强呼吸感，避免大段文字挤在一起造成视觉疲劳；调整了部分样式的色彩表现，提高文字易读性。
- **插画更新**：使用 [Ocean Engine 插画库内容](https://iconpark.oceanengine.com/illustrations/3)，风格更清新。

## 使用

1. 导航到 Obsidian 配置（默认是当前知识库下的 `.obsidian` ）文件夹，找到（或新建）`themes` 文件夹。
2. 克隆 [mark9804/obsidian-border](https://github.com/mark9804/obsidian-border) 仓库到 `themes` 文件夹。文件夹名应为 Arco。
3. 在 Obsidian 设置中启用 `Arco` 主题。
4. 使用 [style settings](https://github.com/mgmeyers/obsidian-style-settings) 插件导入 [Arco.json](https://github.com/mark9804/obsidian-border/blob/theme-arco/presets/Arco.json) 文件。
